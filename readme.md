## Development

Start the development server:

```sh
npm install
```

Start the production server:

```sh
npm start
```

## Project Structure

```sh
src/
├── components/     # API routes
├── lib/            # Utility functions
├── controllers/    # Request handlers
├── models/         # Data models
├── routes/         # API routes
└── app.js          # Express app setup
```
